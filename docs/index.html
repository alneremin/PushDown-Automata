Автомат с магазинной памятью (PushDown Automata)

Язык L = {w: n_a(w) + n_b(w) = n_c(w)} для алфавита A = {a,b,c}

Выражение n_x(w):

  1. n - это число вхождений конкретного символа;
  2. x - это символ, который считается;
  3. w - это цепочка из языка

Ограничения реализации:

  1. Максимальное кол-во символов в магазине - 100;
  2. Множественные эпсилон-переходы не поддерживаются;
  3. Проверка цепочки только по финальному состоянию

Строка принадлежит языку L
